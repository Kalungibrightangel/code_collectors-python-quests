# code_collectors-python-quests

A collection of Python scripts completed as part of our group's programming quests. Each file tackles a different concept, starting from the basics all the way up to functions and logic challenges.

---

## How to run any script

Make sure Python is installed first:
```bash
python3 --version
```

Then just run whichever quest you want:
```bash
python3 quests/quest_29_code_breaker.py
```
